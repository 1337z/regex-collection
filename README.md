## Install

```
npm i regex-collection
```

## Usage
### [Documentation](https://1337z.github.io/regex-collection/)
### Javascript
```javascript
const search = require('regex-collection')
```

### TypeScript
```typescript
import * as search from 'regex-collection'
```

## Supported patterns for:
- Hex color codes
- Telephone numbers
- Email address
